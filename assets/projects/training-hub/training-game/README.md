# Training Game Assets

Drop optional team assets here when you customize the starter map.

Suggested structure:

- `backgrounds/` for map backdrops
- `characters/` for hero sprite sheets
- `checkpoints/` for NPC sprite sheets

These files are copied to `assets/projects/training-hub/training-game/` when you run:

`make -C _projects/training-hub build`

The starter config already points the background to the sample file in `backgrounds/training-floor-placeholder.svg`.
