# Training Game

This sub-project is the editable starter shell for the Training Hub game.

Customize it here instead of editing the shared engine:

- Update `levels/TrainingHubMissionConfig.js` to rename checkpoints, rewrite dialogue, and move items around the map.
- Put optional art in `assets/` when your team wants project-specific backgrounds or sprites.
- Run `make -C _projects/training-hub build` to sync the project files into the public site assets.

Build output:

- JavaScript levels copy to `assets/js/projects/training-hub/training-game/`
- Project assets copy to `assets/projects/training-hub/training-game/`
